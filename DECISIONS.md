# Решения по архитектуре

1. **Slim Framework** — минималистичный, показывает понимание HTTP
2. **SQLite** — соответствует ТЗ, не требует отдельного сервера
3. **Medoo** — лёгкая библиотека для работы с БД
4. **Конкурентность** — решена на уровне SQL (conditional update)
5. **Сессии** — достаточно для демо-проекта
6. **Vanilla JS** — без фреймворков для простоты
